고차방정식 — 인수분해 완결 조건 오개념

잘못된 개념 체계
📊 발생 빈도: 진단 5회 중 4회 반복 ⏱ 예상 교정: 2~3시간 🔗 영향 단원: 이차방정식·삼각함수 등식

핵심 오개념: "인수분해는 공통인수를 묶으면 끝난다"는 잘못된 체계를 갖고 있음. 인수분해의 목적이 '일차식의 곱으로 완전히 분해하여 해의 개수를 확보하는 것'임을 모름. x(x²-1) = 0에서 x²-1이 이차식임을 인식하지 못하고 종료 판단.

선택한 오답: ③ x = 0, 1  →  정답: x = 0, 1, −1 (3개)
잘못된 개념 체계: "더 이상 공통인수가 없으면 완료" → 실제로는 "모든 인수가 일차식이 될 때까지" 분해해야 함
역추적 원인: 인수분해 단원에서 a²−b² = (a+b)(a-b) 공식의 적용 조건을 절차로만 암기, 개념 이해 없음

일차함수 — 함수식 구조(계수 위치) 오개념

잘못된 개념 체계
📊 그래프 해석 문항 정답률 38% ⏱ 예상 교정: 1~2시간 🔗 영향 단원: 이차함수·도형의 방정식

핵심 오개념: y = ax + b에서 a와 b의 역할에 대한 개념 체계가 형성되지 않음. "x 앞에 있는 수가 기울기"라는 구조 인식이 없어 문자식을 숫자 자체로만 인식. 이로 인해 y = 2x + 3에서 기울기=3, y절편=2로 항상 거꾸로 읽는 패턴 반복.

선택한 오답: 기울기 = 3 선택 (y = 3x + 2로 혼동)
잘못된 개념 체계: "계수는 x 앞에 붙는 수"라는 구조 인식 부재 → 첫 번째 수를 기울기로 읽는 위치 오류
역추적 원인: 문자식 단원(ax + b 구조)에서 계수·상수항 구분 개념이 형식적으로만 학습됨

등식의 성질 — 등호(=) 의미 오개념

잘못된 개념 체계
📊 단독 문항 80% vs 복합 문항 58% ⏱ 예상 교정: 1~2시간 🔗 영향 단원: 연립방정식·부등식·이차방정식

핵심 오개념: 등호(=)를 "계산 결과를 쓰는 기호"로 인식 (연산 기호 오개념). 등호의 진짜 의미인 "양쪽이 항상 같아야 한다는 균형 관계"를 이해하지 못함. 이 때문에 이항이 "부호를 반대로 바꿔 반대쪽으로 넘기는 것"이 아니라, "균형 유지를 위해 빼기를 더하기로 바꾸는 것"임을 인식하지 못한다.

오류 패턴: 3x − 5 = 7 → 3x = 7 − 5 (−5를 그대로 우변에 유지)
잘못된 개념 체계: "이항 = 항을 이동하는 절차" → 실제로는 "등호 균형을 유지하는 역연산"
역추적 원인: 초등 산수에서 = 기호를 "답을 쓰는 자리"로 학습한 개념 체계가 고착됨

이차함수 vs 이차방정식 — 수식의 목적 오개념

잘못된 개념 체계
📊 혼합 문항 정답률 52% ⏱ 예상 교정: 2시간 🔗 영향 단원: 판별식·그래프와 직선의 교점

핵심 오개념: 함수(y = ax²+bx+c)와 방정식(ax²+bx+c = 0)이 형태는 비슷하지만 "묻는 것"이 전혀 다르다는 체계 인식 부재. 함수는 "모든 x에 대한 y값의 관계"를, 방정식은 "y=0을 만족하는 특정 x값"을 구하는 것임을 구분하지 못함.

오류 패턴: "x축과의 교점 좌표" 문항에서 y=0 대입 없이 꼭짓점 공식 적용
잘못된 개념 체계: "y가 있으면 함수 공식, 없으면 방정식 공식" → "문제가 묻는 것이 무엇인가"에 따른 목적 기반 구분 체계 미형성
역추적 원인: 이차함수 그래프와 x축 교점의 관계를 시각적으로 연결하지 않고 공식으로만 학습
